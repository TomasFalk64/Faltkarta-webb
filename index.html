Vad appen gör

Fältkarta är en mobilapp för att dokumentera artobservationer direkt i fält, självklart även utan uppkoppling.

Du kan använda egna kartor eller ladda ner från t.ex. Skogsmonitor.se.

När du är färdig exporterar du direkt till Artportalen eller till din dator. Ingen extra konvertering behövs.

Funktioner

  • Enkel registrering: Snabbt formulär med artförslag. Skriv t.ex. bl och välj Blanksvart spiklav.
  • Egna kartor: Importera valfritt antal GeoTIFF.
  • GPS: Justerbar GPS-frekvens och valfri bakgrundspositionering gör att appen behåller kontakten med satelliterna även när skärmen är släckt
  • Bilder: Bilder namnges med art och tidpunkt, vilket underlättar export till Artportalen eller när du vill göra en rapport.
  • Polygoner: Importera eller rita egna inventeringsområden.

Export

  • Artportalen: Direkt utan konvertering
  • OBS kräver SWEREF99 som inställning i Artportalen :
  • Excel: För vidare bearbetning
  • GeoJSON: För QGIS
  • ZIP: Exportera en komprimerad zip-fil med karta + data + bilder och skicka den till din dator.

Manual för Fältkarta

Innan du börjar:

För att kunna skicka dina fynd direkt till Artportalen behöver du:

  • Ett konto på Artportalen.se.
  • Viktigt: Logga in på Artportalen via en dator, gå till Mina sidor > Inställningar och se till att Koordinatsystem är inställt på SWEREF 99 TM.
  • Ladda ner Fältkarta från Google Play eller App Store.

1. Kom igång

  • Öppna appen och följ instruktionerna för att ge nödvändiga tillstånd.
  • GPS-tillåtelse: Ge appen tillåtelse att använda din plats (välj "När appen används").

2. Ladda in karta

Du behöver ladda ner en bakgrundskarta till mobilen, antingen från en extern källa eller genom att skapa en egen.

  • Från QGIS: Skapa din karta. Välj Projekt > Importera/Exportera > Exportera som bild. Justera vid behov upplösning (DPI) och yta. Spara som .tif.
  • Via appen: Klicka på +-knappen i appens huvudmeny.
  • Öppna webbläsaren Tar dig till skogsmonitor.se där du kan ladda ner färdiga kartbilder.
  • Ladda från enhet Välj den sparade kartfilen (.tif) från mobilens minne.
  • Namnge kartan Ge kartan ett tydligt namn. Detta namn används som projektbeteckning vid export.
  • Tips: Ändra namn i efterhand via de tre prickarna bredvid kartans namn.

3. Registrera observationer

  • Navigera: När du öppnat kartan kan du zooma och flytta den. Hårkorset i mitten visar var observationen hamnar.
  • Registrera: Klicka på knappnålen. Börja skriva artnamnet och välj från förslagslistan.
  • Information: Klicka på "Rödlistestatus" för att se artens bevarandestatus.
  • Formulär: Fyll i beskrivning, antal m.m. Spara med grön check-knapp eller radera med röd soptunna.
  • Positionering: Klicka på pil-knappen för att centrera på din plats. Om Följ är aktiverat rör sig kartan med dig.
  • Riktning: Klicka på kompassrosen för att se din färdriktning.

4. Exportera data

  • Gå tillbaka till huvudmenyn, klicka på de tre prickarna bredvid din karta och välj Export.
  • Välj format: Excel, GeoJSON eller ZIP för dator, eller direkt till Artportalen.
  • Export till Artportalen:
    1. Logga in med ditt konto i vyn som öppnas.
    2. Klicka i inmatningsfältet, håll kvar fingret i en sekund och välj Klistra in.
    3. Klicka på den gröna knappen för att skicka.
    4. Viktigt: Du måste fortfarande logga in på Artportalens hemsida efteråt för att bekräfta dina observationer.
